# FinTrack Pro - CLI Finance Manager

FinTrack Pro is a command-line personal finance management system built using Python, SQLite and SQLAlchemy ORM.

## Features
- Add, update, delete expenses
- Category wise spending analytics
- Search expenses by date
- Monthly budget alerts
- Subscription tracker
- Persistent SQLite storage

## Tech Stack
Python, SQLAlchemy ORM, SQLite, Raw SQL, CLI

## Concepts Used
- ORM Relationships
- SQL Joins & Aggregation
- Modular Architecture
- Exception Handling
- Real world business logic
